Data Structures
A data structure is a specialized format for organizing and storing data.
General data structures include:
  • Arrays
  • Stacks
  • Queues
  • Linked Lists
  • Trees
  • Graphs

etc...
We need data structures for two major purposes:
  • Time
  • Space
Time determines how fast a program is. Space determines how much space it is consuming.And needless to say, both are important when it comes to coding, because we all want programs to perform efficiently, fast and with less memory consumption.
